Installazione e requisiti

Requisiti

- Python 3.11 o superiore (consigliato Python 3.13)
- File `rockyou_opt.db` o `rockyou.db` nella stessa cartella di `flaw.py`

Installazione

1. Copia tutti i file del progetto nella stessa cartella.
2. Assicurati di avere Python installato.
3. Esegui il programma con:

   python main.py --help

4. Per usare la versione inglese, esegui:

   python main_eng.py --help

Non sono necessari pacchetti esterni oltre alla libreria standard di Python.
